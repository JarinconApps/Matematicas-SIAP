\subsection*{Palabras Clave}
Mallas
\rule[0.25ex]{1\columnwidth}{1pt}
Resumen:
Este póster presentara un método para la discretización de dominios bidimensionales mediante el código SpecMesh2d basado en el lenguaje fortran, para ello es necesario el estudio de que es una malla estructurada y no-estructurada, conocer algunos de los métodos en lo que se puede generar una malla dependiendo el dominio que se necesite discretizar. Se conocerá un poco de lo que puede ofrecer el código SpecMesh2D en gfortran, para el cual se necesitara un visualizador para las gráficas, que se deben programar en algún editor de código dadas ecuaciones paramétricas, dado esto se creara un archivo el cual visualizaremos en los resultados del póster tanto las características que llevara la discretización como la el dominio que se va a discretar.

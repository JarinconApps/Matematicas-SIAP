\subsection*{Palabras Clave}
Ajuste de Curvas, Mínimos cuadrados
\rule[0.25ex]{1\columnwidth}{1pt}
El asunto a tratar resulta cuando se dispone de una información de datos relacionados, y se desea caracterizar la tendencia de dichos datos a partir de una función. Una manera para determinar la función es inspeccionar en forma visual los datos graficados y después trazar una “mejor” línea a través de los puntos. Aunque tales procedimientos “a ojo” apelan al sentido común y son válidos para cálculos “superficiales”, resultan deficientes por ser arbitrarios. Es decir, a menos que los puntos definan una línea recta perfecta. Aunque este primer paso es de suma importancia para suponer la ecuación de ajuste, se debe encontrar algún criterio para establecer una base para el ajuste. Una forma de hacerlo es obtener una curva que minimice la discrepancia entre los puntos y la curva. Una técnica para lograr tal objetivo es la llamada regresión por mínimos cuadrados.

El presente cursillo pretende ofrecer mediante dos exposiciones, cada una de 90 minutos, una introducción a la técnica de ajuste de curvas por mínimos cuadrados y los temas a considerar son los siguientes:

\begin{enumerate}
\item Regresión lineal 
\item La ecuación de una recta 
\item Linealización de relaciones no lineales 
\item La ecuación exponencial 
\item La ecuación de potencias 
\item La ecuación de razón del crecimiento 
\item La ecuación de sinusoidales
\item Regresión polinomial 
\item Regresión lineal múltiple 
\item Mínimos cuadrados lineales en general 
\item Regresión no lineal 
\end{enumerate}
\textbf{Nota:} Las exposiciones estarán acompañadas con un software
educativo que permitirá ejemplificar cada uno de los ajustes tratados.

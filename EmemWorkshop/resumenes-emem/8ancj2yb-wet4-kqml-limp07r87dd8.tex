\subsection*{Palabras Clave}
Redes Neuronales, Ajuste de Curvas , Predicción de comportamiento 
\rule[0.25ex]{1\columnwidth}{1pt}
A través de la historia la ciencia se ha basado en análisis de datos para general conclusiones y aportar soluciones a nuevas teorias. En este sentido las Redes Neuronales Aportan Mecanismos para el análisis de datos, la predición de comportamientos, la clasificación, que ayudan a generar aportes para la ciencia. 
Una de las problemáticas que existen al rededor del análisis de datos, es el tipo de técnicas matemáticas que se usa para establecer relaciones entre los datos. Una técnica muy conocida es la regresión, dentro de las cuales se encuentra la Regresión lineal, logarítmica, potencial, logística, etc. Estos tipos de técnicas consisten en encontrar una función que se ajuste de la mejor forma posible a unos datos dados, en ocasiones este ajuste no es el mejor. Por otra parte, existe la interpolación que trata de encontrar una curva que tome todos los datos de un experimento. De esta forma podemos encontrar varios métodos y teorías para determinar el ajuste de una curva, en este caso se quiere usar las redes neuronales, ya que el estudio de estas permite establecer relaciones entre los datos, además ofrece otras ventajas como lo son, la clasificación y la predicción de patrones que otros métodos no ofrecen, en este sentido existen fenómenos y experimentos que no obedecen a una clasificación de curvas ya conocidas, por tal razón es necesario usar un método que permita determinar relaciones arbitrarias en tales fenómenos.

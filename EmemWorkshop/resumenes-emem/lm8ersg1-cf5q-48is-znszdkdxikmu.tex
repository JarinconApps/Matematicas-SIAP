\subsection*{Palabras Clave}
Análisis de datos, Regresión, Desarrollo de software
\rule[0.25ex]{1\columnwidth}{1pt}
En este trabajo de grado se diseñará una aplicación en la plataforma Delphi para el análisis datos usando regresión, la idea surge al observar que en varios softwares como SPSS (Norman H. Dale H. 1975), Stata (StataCorp. 1985), R-Project (Ross I. Gentleman R.), HidroEsta (Villón M), el usuario ingresa los datos y obtiene el modelo de regresión sin un buen análisis de datos, además en la mayoría de los casos una de las problemáticas es la importación de los mismos datos, por ejemplo, se deben introducir manualmente o en otros casos se deben exportar a formatos poco conocidos para los usuarios.
En si lo que se pretende es hacer una aplicación que sea amigable para el usuario al momento de ingresar los datos obtenidos de cualquier experimento, ya ingresados los datos se tendría la opción para que el usuario pueda exportar todo el análisis a un archivo de Word, cuyo archivo es adaptado para que el usuario lo pueda anexar en su trabajo de grado, tesis, etc. 
Se planea implementar en primer lugar la regresión lineal, luego sigue la regresión cuadrática, exponencial, logarítmica, polinómica y logística. Al momento de culminar cada regresión y obtener algunos datos como ejemplo, se compararán con los datos obtenidos con otras aplicaciones de la comunidad científica para optimizar tanto el análisis de datos como la interfaz de la aplicación.
